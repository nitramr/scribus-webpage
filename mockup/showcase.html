Tasty Baked

A new project with the use of Scribus. I had a great opportunity, starting with a photoshoot where I follow the baker in order to proces his tasty bread. After shooting and retouching those images they asked me to design a large image with a logo and text on it. I used Scribus v1.5.1svn and send this large design to a local printshop. This time I had the change to follow this proces in person. For me it was also a great experience to see how this proces leads to a high quality wall presentation.

Manufacturing Artwork

We did a project for a big printing company who created hardware for one of there printer series. The main focus was to create a full set of inspirational art concerning: Manuals, serie number stickers and single color package print. Important question before we start was if it has to be printed offset or not. I Choose to start this project again with Scribus V1.4.3 and later V1.4.4 It did work out very very well.
