New Text Layout Engine with Full OpenType Support

As of 4 December 2016, Scribus 1.5.3svn provides a new text layout engine with support for OpenType advanced features, as well as complex and right-to-left scripts. For text features, a separate “Text Properties” palette has been created. Happy testing!

Krita Import Filter Added to Trunk

As of revision 21427, Scribus 1.5.3svn includes an import filter for Krita files (KRA). Happy testing!

Scribus 1.5.2 Released

[17 May 2016] The Scribus Team is very pleased to announce the release of the development version Scribus 1.5.2, which is another major step onto the next fully-featured version 1.6.0. In terms of stability, 1.5.2 can be already be regarded as robust.

ScribusCTL project started

Veteran Scribus developer Andreas Vox has started the ScribusCTL project. CTL stands for complex text layout and attempts, among others, to integrate the OpenType text shaping engine HarfBuzz. ScribusCTL will be synchronized with Scribus 1.5.1svn, and every finished project stage will be integrated into the official Scribus 1.5.1svn branch.
