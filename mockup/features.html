Features

Advanced Text Formatting

  • Character and paragraph styles
  • Manual tracking for accurate typesetting
  • Baseline grid aligns text across frames
  • Optical margins
  • Font management and preview

Graphics Tools

  • Rich set of vector editing tools
  • Gradients and transparencies
  • Bitmap image effects
  • Mark-up rendering (e.g. LaTeX)
  • Alignment and distribution tools
  • Image manager

Page Management

  • Master pages allow for standardisation
  • Pages can be easily re-ordered
  • Import other documents
  • Unlimited layers

Output and Printing

  • Pre-flight verifier to check output
  • PDF 1.4 to PDF X/3 supported
  • Bleeds and printer marks (optional)
  • Automatic image compression (optional)
  • PDF security (optional)
  • Resource collection for transfer between machines

Colour Management

  • CMYK and Colour Separation
  • Spot Colours
  • Colour blindness emulation

Extensible

  • Built-in Python interpreter
  • Scripting with Javascript
  • PDF forms supported
  • Can be run from the command line

More...

  • Compatible with most bitmap image formats
  • Supports SVG graphics
  • Supports PDF import
  • Automatic table of contents generation
  • Native XML document format
