Features

Advanced Text Formatting

  • Character and paragraph styles
  • Manual tracking for accurate typesetting
  • Baseline grid aligns text across frames
  • Optical margins
  • Font management and preview

Vector and Object Editing Tools

  • Bezier Curves
  • Primitive Shapes
  • Create Advanced Shapes by Generators

Layout Tool

  • Template and Asset Library
  • Unlimited layers
  • Master pages allow for standardisation
  • Alignment and distribution tools
  • Grids and Guides

Cross Platform working

  • Runs on most common operation systems, like Windows, Mac OS, Linux and others
  • Open file format for documents
  • Integration of Third Party software, like Gimp or Photoshop

Color Management

  • Color Spaces (RGB, CMYK, Lab, Spot Colors)
  • Color Separation
  • Gradients & Fill Patterns
  • Color Palettes
  • Color Blindness Simulation

Output and Printing

  • Pre-flight verifier to check output
  • PDF 1.4 to PDF X/3 supported
  • Bleeds and printer marks (optional)
  • Automatic image compression (optional)
  • PDF security (optional)
  • Resource collection for transfer between machines

Impressive File support

  • Importer: idml, cdr, ai, pub, pmd, svg, emf and more
  • Exporter: pdf, svg, eps, image formats and more
  • Image files: jpg, png, tiff, bmp, psd, kra and more

Non-Destructive Effects

  • Image Effects (Blur, Grayscale, Color Tint, Contrast, etc.)
  • Object Effects (Contour Styles, Dropshadow, Transparency)

Customizable

  • Flexible Workspace
  • Scripter for automations
  • Plugin support
